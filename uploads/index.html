TruthLens

An website for Social Good project designed to combat misinformation, identity theft, and fraud by detecting synthetic media in real-time.

Try The Detector

The Challenge We're Solving

The rise of synthetic media has made it increasingly difficult to distinguish between authentic and manipulated content. Deepfakes threaten individuals and communities by enabling the spread of misinformation, reputational damage, and political propaganda. There is a critical need for tools that can flag deepfakes to protect our digital integrity.

Our Proposed Solution

We propose "TruthLens," an AI-powered system that detects deepfake videos and images in real-time. It uses a lightweight convolutional neural network (CNN) and long short-term memory (LSTM) model to analyze facial inconsistencies and manipulation artifacts. The solution is accessible through a user-friendly web interface and an API for platform integration.

Key Features

Real-Time Detection

Detects deepfake videos and images in real-time.

Explainable Results

Provides explainable heatmaps showing tampered regions of the media.

Platform Integration

Can be integrated with third-party apps or social media via a REST API.

Optimized & Private

The model is optimized for low-resource environments and offers a privacy-focused local deployment option.

Technology and Workflow

Technology Stack

  • Model: TensorFlow, Keras, CNN-LSTM Architecture
  • Pipeline: OpenCV, MediaPipe
  • Web Interface: Flask and ReactJs
  • Dataset: Deepfake Detection Challenge, Celeb-DF
  • Visualization: Grad-CAM for visual explanations

Detection Flowchart

  • User inputs a video or image through the web interface.
  • The Backend Flask API receives the content.
  • A preprocessing module prepares the data for analysis.
  • The Deepfake Detection Model analyzes the content.
  • A prediction output and heatmap visualization are generated.
  • The final results are sent back to the user.

Impact and Future Scope

Potential Real-World Impact

  • Enables journalists and fact-checkers to verify content.
  • Empowers individuals to assess the authenticity of media they encounter.
  • Provides critical tools for law enforcement and digital forensics.
  • Reduces the spread of harmful misinformation on social platforms.

Future Scope

  • Integration with Android and iOS for mobile verification.
  • Extension to detect deepfake audio and synthetic voices.
  • Use behavioral facial recognition to enhance model accuracy.
